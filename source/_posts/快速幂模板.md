### 模板如下

注意，这里添加了取模操作，如果不需要取模，请删除取模运算即可。<!-- more -->

```java
int qmi(int p , int k){

    p %= N ;       //这句话忘了，记得下次写上
    int res  = 1 ;
    while(k){
       if(k & 1)  res = res *  p % N;
       p = p * p % N ;
       k >>= 1 ;
    }
    return res ;
 }
```
